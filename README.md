.gitignore 파일에 프로젝트명을 붙여주면 됨

ex)

## <LFS관련>
100mb 이상 큰 파일은 git lfs사용해야함

1. git lfs install (깃 초기화)
2. git lfs track "*.psd"
3. .gitattributes 생성됨
4. .gitattributes 을 수정
